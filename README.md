# dotfiles

Catppuccin Mocha everywhere. Fedora + GNOME.

![Catppuccin Mocha](https://raw.githubusercontent.com/catppuccin/catppuccin/main/assets/palette/macchiato.png)

## What's inside

| Tool | Config |
|------|--------|
| zsh + oh-my-zsh | `zshrc` |
| starship prompt | `config/starship/` |
| alacritty | `config/alacritty/` |
| bat + delta | `config/bat/`, `git/` |
| GTK 3/4 (libadwaita) | `config/gtk-*/` |
| micro editor | `config/micro/` |
| gamemode | `config/gamemode/` |

## Install

```
git clone https://github.com/YOUR_USER/dotfiles ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

Log out and back in.

## Dependencies (installed automatically)

`zsh` `eza` `bat` `fd-find` `ripgrep` `fzf` `zoxide` `delta` `starship` `alacritty` `fastfetch` `micro` `papirus-icon-theme` `gnome-tweaks` `ImageMagick`
